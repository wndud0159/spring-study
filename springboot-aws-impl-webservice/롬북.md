- @Getter
  - 선언된 모든 필드의 get메소드 생성
- @Setter
  - 선언된 모든 필드의 set메소드 생성
- @RequireArgsConstructor
  - 선언된 모든 final필드가 포함된 생성자를 생성
  - final이 없는 필드는 생성자에 포함되지 않는다
- @NoArgsCOnstructor
  - 기본 생성자를 자동 추가
  - public 클래시명() {} 과 같은 효과
- @Builder
  - 해당 클래스의 빌더 패턴 클래스를 생성
  - 생성자 상단에 선언 시 생성자에 포함된 필드만 빌드에 포함
